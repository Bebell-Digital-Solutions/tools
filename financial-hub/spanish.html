📋 Plantilla Financiera de Notion - Soporte DOP Mejorado

Plantilla Financiera Mejorada con Soporte para DOP

Plantilla completa de Notion con soporte para la moneda del Peso Dominicano (DOP) y fórmulas de transacciones recurrentes

🎯 Qué Incluye:

  • Configuración completa de la base de datos con formato de moneda DOP (RD$)
  • Fórmulas de transacciones recurrentes usando funciones dateAdd
  • Seguimiento de presupuesto con indicadores visuales de progreso
  • Informes financieros mensuales y anuales
  • Seguimiento de metas y gestión de hitos
  • Sistema de categorización de gastos
  • Plantillas de análisis de ingresos y gastos
  • Legibilidad de fórmulas mejorada con codificación por colores

🚀 Características Mejoradas:

  • Propiedades de la base de datos estructuradas en tablas organizadas
  • Cálculos automáticos de transacciones recurrentes
  • Formato de moneda del Peso Dominicano (DOP) en todo
  • Jerarquía visual con markdown mejorado
  • Soporte de codificación por colores para una identificación rápida
  • Indicadores con emojis para un escaneo rápido

📋 Fórmulas de la Base de Datos:

# Fórmula de Presupuesto Restante if(prop("Presupuesto Mensual") > 0, prop("Presupuesto Mensual") - prop("Gasto Actual"), 0) # Fórmula de Progreso del Presupuesto if(prop("Presupuesto Mensual") > 0, round(prop("Gasto Actual") / prop("Presupuesto Mensual") * 100), 0) # Fórmula de Días Hasta el Objetivo dateBetween(prop("Fecha Objetivo"), now(), "days") # Fórmula de Ahorro Mensual Requerido if(prop("Fecha Objetivo") > now(), (prop("Monto Objetivo") - prop("Monto Actual")) / dateBetween(prop("Fecha Objetivo"), now(), "months"), 0) # Fórmula de Transacción Recurrente dateAdd(prop("Fecha de Último Pago"), prop("Frecuencia"), "months")

💰 Configuración de Moneda:

Configuración del Peso Dominicano (DOP):

  • • Símbolo de Moneda: RD$ (Peso Dominicano)
  • • Formato: RD$ #,##0.00
  • • Todas las propiedades financieras configuradas para DOP
  • • Fórmulas de cálculo automático incluidas

🔄 Configuración de Transacciones Recurrentes:

  1. Crear una propiedad "Frecuencia" (selección: Mensual, Semanal, Diario, Anual)
  2. Añadir propiedad "Fecha de Último Pago" (formato de fecha)
  3. Añadir "Próxima Fecha de Vencimiento" con fórmula: dateAdd(prop("Fecha de Último Pago"), 1, prop("Frecuencia"))
  4. Usar automatización para actualizar fechas cuando se registren transacciones

✨ Mejoras Clave:

  • • Estructuración de todas las propiedades de la base de datos en tablas organizadas
  • • Adición de fórmula de transacción recurrente usando dateAdd
  • • Especificación del formato de moneda DOP en todo
  • • Mejora de la jerarquía visual con markdown
  • • Mejora de la legibilidad de las fórmulas
  • • Adición de notas de soporte para codificación por colores
  • • Adición de indicadores con emojis para un escaneo rápido
